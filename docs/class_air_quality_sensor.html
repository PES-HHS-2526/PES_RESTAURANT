Detailed Description

Klasse voor het initialiseren en uitlezen van de SGP30-luchtkwaliteitssensor.

De AirQualitySensor-klasse beheert de communicatie met de SGP30-sensor. De klasse initialiseert de sensor via I2C, voert metingen uit en slaat de laatst gemeten eCO2- en TVOC-waarden op.

Binnen het PES restaurantproject wordt vooral de eCO2-waarde gebruikt om te bepalen of de ventilatie ingeschakeld of uitgeschakeld moet worden.

bool AirQualitySensor::begin ( )

Initialiseert de I2C-verbinding en de SGP30-sensor.

Initialiseert de I2C-bus en de SGP30-sensor.

Returns
true als de sensor correct gestart is, false als initialisatie mislukt.

Deze functie start de I2C-communicatie op de Wemos D1 Mini met SDA op D2 en SCL op D1. Daarna wordt gecontroleerd of de SGP30-sensor correct gevonden en gestart kan worden.

Returns
true als de SGP30-sensor succesvol gestart is, false als de sensor niet gevonden of niet correct geïnitialiseerd kan worden.
